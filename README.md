# ☕ Coffee & Code | CRUD Blog

Сучасний, мінімалістичний та затишний веб-додаток для ведення нотаток та блог-постів. Проєкт створений з акцентом на простоту використання та теплу "кавову" UI/UX естетику.

## 🌟 Про проєкт

"Coffee & Code" — це Single Page Application (SPA), яке демонструє роботу базових операцій роботи з даними (CRUD) без використання зовнішніх серверів чи баз даних. Усі записи зберігаються локально у вашому браузері за допомогою Web Storage API, що гарантує миттєвий відгук інтерфейсу.

## 🛠 Функціонал (CRUD)

- **Create (Створення):** Додавання нових записів із заголовком та основним текстом.
- **Read (Читання):** Відображення стрічки постів у хронологічному порядку (найновіші зверху) з автоматичною фіксацією дати.
- **Update (Оновлення):** Можливість швидкого редагування існуючих записів прямо на сторінці.
- **Delete (Видалення):** Безпечне видалення постів із вбудованим механізмом підтвердження (щоб уникнути випадкових кліків).
- **Data Persistence:** Збереження стану додатку після закриття вкладки або оновлення сторінки завдяки `localStorage`.

## 💻 Технологічний стек

Проєкт побудований виключно на нативних веб-технологіях без використання важких фреймворків чи бібліотек:

* **HTML5** — семантична структура сторінки.
* **CSS3** — кастомний дизайн, Flexbox-верстка, змінні (CSS variables) для керування палітрою кольорів, інтерактивні hover-ефекти.
* **Vanilla JavaScript (ES6+)** — обробка подій, маніпуляції з DOM-деревом, робота з форматами JSON.
